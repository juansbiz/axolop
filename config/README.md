# CRM Configuration Files

This directory contains all configuration files for the application.

## Contents

- **Development Configuration:**
  - `vite.config.js` - Frontend build configuration
  - `eslint.config.js` - Code linter configuration
  
- **TypeScript Configuration:**
  - `tsconfig.json` - TypeScript compiler options
  - `tsconfig.node.json` - Node-specific compiler options
  
- **Styling Configuration:**
  - `tailwind.config.js` - Tailwind CSS configuration
  - `postcss.config.js` - PostCSS processing configuration
  
## Configuration Philosophy

Configuration files follow these principles:
- Environment-specific settings
- Consistent formatting across files
- Clear documentation for options
- Separation of build and runtime configs
